# PitStop Toolkit

Bash scripts I hold onto here to configure tools I repeatedly install on new builds. 
Get up and go, just like a pit stop. I mainly use Debian-based distros.

<ins>**Scripts currently held**</ins>:  
**install_docker.sh**: A quick installer for Docker on Ubuntu 24.04  
**puppy.sh**: An installer/configurer for gmail accounts onto the mutt email client.  
**[vvireguard](https://github.com/mrbillyblack/vvireguard)**: A Terraform configuration that deploys a WireGuard VPN server onto GCP or AWS, with a specified amount of clients. Complete with GitHub Action to streamline the process.  

Navigate to each script and activate with the following:
```
sudo chmod +x script.sh
./script.sh
```
